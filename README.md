# NewLetterEWU

A web-based Newsletter Management System designed to help East West University manage digital newsletters efficiently. This platform allows students, faculty, and administrators to collaboratively create, edit, and publish content such as news, events, research highlights, and announcements.

## Features
- Create, edit, and categorize newsletter content
- Manage events, achievements, and research publications
- Admin and faculty login system
- Responsive front-end for mobile and desktop users
- Easy collaboration and content review

## Technologies Used
- HTML, CSS, JavaScript
- Node.js (if applicable)
- Git & GitHub for version control
